# Android Gold Icon — mipmap folders

Place `ic_launcher_gold.png` in each mipmap density folder:

| Folder           | Size (px) |
|------------------|-----------|
| mipmap-mdpi      | 48×48     |
| mipmap-hdpi      | 72×72     |
| mipmap-xhdpi     | 96×96     |
| mipmap-xxhdpi    | 144×144   |
| mipmap-xxxhdpi   | 192×192   |

The filename must be `ic_launcher_gold.png` — matching `@mipmap/ic_launcher_gold`
referenced in the `MainActivityGoldIcon` activity-alias in AndroidManifest.xml.
